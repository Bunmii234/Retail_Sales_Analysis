📌 Project Overview
This project focuses on transforming raw retail transaction data into actionable business insights. By analyzing 1,000 unique customer interactions, I identified key revenue drivers across different age demographics and product categories.

🎯 Objective
To demonstrate full-stack data proficiency by:
1. Cleaning & Transforming data using SQL.
2. Analyzing trends via Excel Pivot Tables.
3. Presenting findings through a custom-built HTML/CSS portfolio.


 🛠️ Technical Stack
1. SQL: Used for data aggregation, filtering by age group, and calculating total revenue per category.
2. Excel: Developed a dynamic dashboard for visual reporting.
3. Web Dev: HTML5 and CSS3 for professional portfolio presentation.



💡 Key Insights
| Demographic | Top Performing Category | Total Revenue |

| Ages 41-60 | Electronics | $64,635 |
| Ages 25-40 | Clothing | $56,480 |
| Ages 18-24 | Beauty | $26,020 |
